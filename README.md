# 🌊 Flow: The Dynamic Island App

## ✨ What it does
Flow is a uniquely Apple-focused iOS application leveraging the Dynamic Island for a beautiful and intuitive user experience. It demonstrates advanced modern iOS development featuring rich animations and deep system integration.

## 🛠 Tech Stack
![Swift](https://img.shields.io/badge/Swift-FA7343?style=for-the-badge&logo=swift&logoColor=white) ![SwiftUI](https://img.shields.io/badge/SwiftUI-007AFF?style=for-the-badge&logo=swift&logoColor=white) ![iOS](https://img.shields.io/badge/iOS-000000?style=for-the-badge&logo=apple&logoColor=white)

## 🚀 Key Features
- **Dynamic Island Integration**: Seamless live activities and real-time updates directly in the Dynamic Island
- **Rich Animations**: Fluid, state-driven animations using SwiftUI
- **Native Experience**: Feels like a built-in iOS application

## 📸 Demo
See Flow in action! Download the source code, open in Xcode 15+, and build for an iPhone 14 Pro or newer simulator to experience the Dynamic Island natively.

```bash
git clone https://github.com/Ripnrip/Flow.git
```
